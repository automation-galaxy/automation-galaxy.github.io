Customise vRO Environment Add custom .py files

  1. Create a vRO package (must be vRO 8.8 or above).
  2. Add the vRO environment that you want to add custom files to.
  3. Save and Export the vRO package.
  4. Extract the contents of the package file (used 7-Zip on Windows for testing).
  5. Browse the exported folders and find the bundle file in the elements->UUID-> folder.
  6. Extract the bundle file with 7-Zip (most compression utilities will fail to extract due to how the archive was created).
  7. Browse the extracted folder to the lib folder, place your custom Python files in here.
  8. Create a zip from the environment-bundle folder. The environment-bundle folder should be added to the zip, not just its contents.
  9. Rename the created archive to bundle and replace the old bundle file with this newly created file. Make sure to remove any other extracted files from the folder hierarchy.
  10. Now, in the root of the extracted package folder (the folder that contains certificates, elements, and signature folders, along with dunes-meta-inf), create a zip file with the 3 folders and the duens-meta file. This folder should be renamed to a .package file and imported back into vRO.
  11. The vRO environment should now be updated with the custom files.
